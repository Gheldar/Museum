INSTALLATION INSTRUCTIONS:


1) Unrar and copy the contents of opensim to a folder of your choice

2) Download and setup OpenSim by running bin/Opensim.exe

3) Once OpenSim is ready, run the command: load oar [oar's directory without brackets]

4) Create an account with the command: create user

4) Download and install Firestorm viewer

5) Run Firestorm viewer and log in with the account created

6) Enjoy!
